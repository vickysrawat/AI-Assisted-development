# Migration 017 — v3.7.0

## Plugin source directory consolidation

The plugin's deployment source directories have been reorganised into a single
`_project-deploy/` root:

| Old location | New location |
|---|---|
| `hooks/` | `_project-deploy/hooks/` |
| `rules/` | `_project-deploy/rules/` |
| `skills/command-stubs/` | `_project-deploy/commands/` |

**This has no impact on target projects.** Only the plugin's internal source structure
changed. Files deployed to `.claude/hooks/`, `.claude/rules/`, and `.claude/commands/`
in target projects are identical; only the source path the bootstrap reads from has moved.

No action required in target projects. Run `/dream-sync` as usual after upgrading.
